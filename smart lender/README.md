# Smart Lender - AI Based Loan Approval Prediction

## Project Description

Smart Lender is a web application developed using Python and Flask that predicts whether a loan application is likely to be approved or rejected based on applicant details.

This project demonstrates the basic workflow of a loan approval prediction system with a user-friendly interface.

## Features

- Loan Approval Prediction
- Professional User Interface
- Fast Response
- Flask Web Application
- Simple Decision Logic

## Technologies Used

- Python
- Flask
- HTML
- CSS

## How to Run

1. Install Flask

```
pip install flask
```

2. Run the application

```
python app.py
```

3. Open Browser

```
http://127.0.0.1:5000
```
